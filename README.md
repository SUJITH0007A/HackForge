# HackForge

HackForge is a production-style, event operating system for hackathons, datathons, coding contests, and college technical events.

## Structure

- `backend/`: Express + MongoDB + Mongoose + JWT + Socket.io
- `frontend/`: React + Tailwind CSS + Framer Motion + Recharts

## Local

1. `npm install`
2. Set env vars (see `backend/.env.example` and `frontend/.env.example`)
3. `npm run dev`
